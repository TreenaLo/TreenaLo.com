BAO_BI Tea Package





Overview


Project Type:

Personal work


Goal:

Create a package set for BOBA Tea. The set should
have one big box and three small iron boxes.
The package needs to relate to Chinese style and
present fresh.


Role:

Graphic Designer


Software used:

Illustrator, Photoshop


Challenge:

There are many packages about tea leaves in the
market, so how to design an outstanding package to
grab consumer attention is a challenge.


Solution:

When people think about tea leaves, they will think of
green and brown because these colours are related to
tea. However, I use yellow for packaging sleeves to
increase the visual contrast effect.
